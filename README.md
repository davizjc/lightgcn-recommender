# lightgcn-recommender

A recommendation system built with **LightGCN** — a graph convolutional network
for collaborative filtering that learns user and item embeddings by propagating
them over the user-item interaction graph.

## Contents

- `LightGCN.ipynb` — model implementation, training loop, and evaluation.

## Run

```bash
jupyter notebook LightGCN.ipynb
```

## Tech

Python, PyTorch, graph neural networks, recommender systems.

## Reference

He et al., *LightGCN: Simplifying and Powering Graph Convolution Network for
Recommendation* (SIGIR 2020).
